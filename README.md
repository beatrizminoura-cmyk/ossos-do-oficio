# Ossos-do-Ofício
  Ossos do Ofício é um jogo desenvolvido no RPG Maker MV, cujo objetivo é proporcionar uma experiência narrativa e interativa por meio de mecânicas simples e visual em estilo 2D.
O projeto criado é uma combinação de um quiz matemático junto com aventura, que tem como tema principal a matemática e a alquimia e possui três fases no total.

  As pastas e arquivos do jogo estão na seção "main" do repositório, são eles: pastas - audio, data, fonts, icon, img, js e arquivos - GALV_EventDetectors, index.html, lily.json, Maitê.json, principal.json, package.json.
